API de spaintvs

Para crear tu front-end


PyDownTV web, utiliza la biblioteca Python de spaintvs para obtener los enlaces de descarga de los vídeos de los canales de televisión. En la página del proyecto se están desarrollando aplicaciones para Escritorio (C++ & QT5) y Android (Java Android SDK), pero si quieres crear tus propias aplicaciones y te apetece solo centrarte en la vista puedes utilizar nuestra API Online

A la mini API se accede desde esta url () y haciendo peticiones HTTP GET a "url".
(También existe una manera más "RESTfuliana": /[AQUÍ LA URL])

La API devolverá respuestas en formato JSON con la info del vídeo pedido.
