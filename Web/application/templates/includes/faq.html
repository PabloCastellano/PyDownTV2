Preguntas Frecuentes


¿Qué es exactamente PyDownTV?

PyDownTV es un proyecto que se encuntra en desarrollo en GitHub que consta de una serie de Apps para ayudar a descargar los vídeos de las páginas web de las principales cadenas de televisión españolas. PyDownTV web (esta web) es una de ellas.

¿Funciona PyDownTV Web en cualquier navegador y/o Sistema Operativo?

Debería ;) Aunque la mejor experiencia de usuario la obtendrás con Google Chrome/Chromium o Safari
PyDownTV está diseñado para que funcione en cualquier navegador y en cualquier Sistema Operativo, incluso para adaptarse a todos los tamaños de pantalla tanto de ordenadores personales como tablets o smartphones. Sin embargo, está optimizado para funcionar de manera correcta en Google Chrome/Chromium.
Se han detectado, por ejemplo, fallos de maquetación en alguna versión de Firefox e Internet Explorer.
Para el correcto funcionamiento de PyDownTV el navegador debe tener Javascript activado (esto es lo normal) y es preferible que se use un navegador actualizado a su última versión.
Desde PyDownTV se recomienda utilizar Google Chrome/Chromium para navegar por la aplicación y obtener así el máximo redimiento de la misma.

¿Cómo puedo descargar un vídeo a través de esta web?

Muy fácil, solo tienes que sguir los pasos indicados más abajo, pero por si acaso, los detallo más a continuación.
Básicamente, si estás en una navegando por la web de una televisión de España, y estás justo en una página donde hay un vídeo, lo único que tienes que hacer es copiar la URL (la dirección web, esa que empieza por "http://www...") que se encuentra en la barra de navegación. Después tendrás que entrar en esta web, y en el cuadro de búsqueda que se encuentra arriba de todo, tendrás que pegar el enlace copiado antes. Una vez hecho esto, solo hay que dar a Buscar! y esperar a que PyDownTV encuentre el vídeo.
Si PyDownTV encuentra el vídeo, entonces solo tendrás que dar sobre el enlace de las partes del mismo con clic derecho de ratón y darle a "Guardar enlace como" a cada una de las partes. Hay vídeos que no se pueden descargar con este procedimiento, para ello necesitaremos un programa especial.

¿Y cómo se descargan los vídeos "rtmp"?

(Para Descargar con Easy-Rtmpdump o RTMP-Downloader mirad la siguiente FAQ ;))

Esos vídeos que aparecen como "rtmp", no se pueden descargar directamente haciendo clic con el botón derecho del ratón. Necesitarás un programa especial. En GitHub estamos desarrollando una versión de Escritorio sobre todo para ayudar en la descarga de estos vídeos. Pero mientras, lo mejor es utilizar un programa llamado rtmpdump.
Lo único malo es que rtmpdump no dispone de interfaz gráfica, y es un programa que se ejecuta desde línea de comandos, pero vamos a ello, los pasos a seguir son:

  1. Descarga rtmpdump desde aquí:
    • Versión para Windows. No es un instalador, ver siguiente paso...
    • Versión para Mac OS X o instalar a través de Homebrew.
    • Para GNU/Linux, instalar a través de el gestor de paquetes (ejemplo Debian: sudo apt-get install rtmpdump)
  2. Ejecutar rtmpdump:
    • Mac OS X: Abrir la Terminal (Aplicaciones > Utilidades > Terminal) y pegar el comando que te propone PyDownTV. El vídeo se descargará en tu carpeta personal.
    • GNU/Linux: Se supone que el que usa GNU/Linux ya tiene más conocimiento sobre la terminal, pero por si acaso, el procedimiento, es el mismo que para Mac OS X: Abrir terminal, copiar comando, ENTER.
    • Windows: Aquí ya tenemos más problemas, por que normalmente al usuario de Windows le da más miedo la Consola. Hay varias maneras más correctas de "instalar" rtmpdump que la siguiente, pero quizá sea esta la más fácil.
      • Copiar el archivo rtmpdump.exe descargado y pegar este en la carpeta System32 de tu sistema. Quedará entonces el ejecutable en la siguiente ruta (NOTA: puede cambiar la unidad C:, D:, ..): c:\Windows\System32\rtmpdump.exe
      • A partir de ahora solo tienes que Abrir la consola y pegar el comando que proporiciona PyDownTV para bajar el vídeo. Para abrir la consola, hay varias maneras, una es Inicio > ejecutar, y escribir "cmd" y presionar ENTER (o Inicio y en el cuadro de búsqueda escribir "cmd" para versiones recientes de Windows). Para pegar en la consola de Windows, se hace clic con el botón derecho del ratón sobre el fondo negro de la consola, y se selecciona pegar.
      • El vídeo se descargará en la raíz del sistema. Se podrá acceder a el a través de Mi PC > Disco Local (C:) (puede cambiar la Unidad) para versiones de Windows igual o menores que Windows XP. Y para versiones Windows Vista y superirores, se podrá acceder al vídeo a través de Equipo > Disco Local (C:) (puede cambiar la unidad).

¿Cómo descargo los rtmp con Easy-Rtmpdump y/o RTMP-Downloader

Lo primero es lo primero, Easy-rtmpdump es un proyecto distinto a PyDownTV, aunque está diseñado para él por los mismos desarrolladores (también es Software Libre y su código se puede ver en Github). Solo está disponible para Mac OS X y GNU/Linux por ahora. ¿Por qué? Porque @forest de descargavideos.tv ya tiene una buena app para este Sistema Operativo: RTMP-Downloader.

  • Easy-rtmpdump
    • Mac OS X Requisitos:
      1. Python: preinstalado en el sistema operativo
      2. Instalar PIP: Abrir terminal y pegar lo siguiente
        3. curl https://raw.github.com/pypa/pip/master/contrib/get-pip.py | python
      3. Instalar gevent-socketio: Abrir terminal y pegar lo siguiente: 
        4. sudo pip install gevent-socketio
      4. Descargar Easy-rtmpdump de aquí: Descargar Easy-rtmpdump
      5. Descomprimir el archivo .zip donde se quiera. Y ya no hay más pasos ;)
      6. Para ejecutar el programa, doble clic en: Easy-rtmpdump_MACOSX. Se abre una pestaña en el navegador (recomendado Google Chrome) donde se interactuará con el programa.
      7. Ya solo tienes que obtener los enlaces con pydowntv y darle a descargar con Easy-Rtmpdump ;)
      8. Ver vídeo explicativo:
    • GNU/Linux Requisitos:
      1. Python: preinstalado en el sistema operativo
      2. Instalar PIP, Python-greenlet, Python-pip, Python-dev, Python-tk y rtmpdump (Ejemplo Debian/Ubuntu):
        3. sudo apt-get install python-greenlet python-pip python-dev python-tk rtmpdump
      3. Instalar gevent-socketio:
        4. sudo pip install gevent-socketio
      4. Descargar Easy-rtmpdump de aquí: Descargar Easy-rtmpdump
      5. Descomprimir el archivo .zip donde se quiera. Y ya no hay más pasos ;)
      6. Para ejecutar el programa, doble clic en: Easy-rtmpdump_LINUX_32bits o Easy-rtmpdump_LINUX_64bits. Se abre una pestaña en el navegador (recomendado Google Chrome) donde se interactuará con el programa.
      7. Ya solo tienes que obtener los enlaces con pydowntv y darle a descargar con Easy-Rtmpdump ;)
      8. Ver vídeo explicativo: Ver vídeo Mac OS X
  • RTMP-Downloader
    • Windows
      1. Descargar el programa: Versión 0.2b Otras Versiones
      2. Pasos a seguir:
        • Descargar el archivo con el programa
        • Descomprimir el archivo en una carpeta. Esta carpeta será donde se guarden también los vídeos
        • Abrir el programa haciendo doble clic en el ejecutable (RTMP-Downloader.exe)
        • Se abrirá una pestaña/ventana nueva en el navegador. No cerrarla.
        • Buscar los enlaces con pydowntv
        • Hacer clic en descargar con RTMP-Downloader (Windows)
        • Esperar a que la descarga termine (se puede ver en la web que abre el programa al iniciarse)
        • Se abrirá una carpeta con el vídeo una vez descargado.

PyDownTV web no funciona para un enlace que estoy seguro que es correcto, ¿Qué hago?

PyDownTV no es infalible a la hora de recoger toda la información sobre los vídeos, por lo que puede fallar, y lo hará. Lo importante, es que si detectas un fallo en la aplicación, nos lo hagas saber, para poder resolverlo cuanto antes. Si encuentras un problema con la web, por favor, háznoslo saber en el siguiente enlace: Reportar errores PyDownTV.

  1. Crea una nueva Issue, haciendo clic a la derecha arriba en New Issue.
  2. En la derecha especifica que es un bug.
  3. Elige el título que creas conveniente, pero que sea descriptivo y que se espeifique que se hace alusión a la versión web de PyDownTV.
  4. Rellena el cuadro de la Issue con las particularidades del fallo, intentando explicar lo mejor posible todos los detalles.

Mi Tele descarga un archivo de poco tamaño en vez del vídeo

Antes que nada, los problemas del tiempo de descarga con MiTele deberían estar solucionados gracias a una GENIAL idea de el javiero, cuya web podéis visitar aquí. Gracias de nuevo, a veces, las mejores ideas, son las más simples ;).

Por si acaso, dejo la información del error que daba antes a continuación:

No pasa nada, vas a poder descargarlos, pero tienes que ser rápid@ ;)
Los vídeos de MiTele, al generarse el enlace para aser descargados desde el servidor de MiTele, sino se accede a ellos en poco segundos caducan, es por eso, que solo se descarga un archivo (.html) de poco tamaño.

Para porder descargar los vídeos de MiTele hay que ser rápidos. Tan rápidos, que no da tiempo a hacer clic derecho con el ratón para guardarlo y descargarlo.
Por eso propongo unas opciones:

  • Usando Chrome: tienes que hacer un simple clic normal con el ratón, pero mientras mantienes presionada la tecla "Alt" del teclado.
    La tecla "Alt" es una tecla que suele estar situada en la primera fila de los teclados, empezando por abajo. Suele estar tanto a la derecha como a la izquierda de la barra espaciadora, justo después de la tecla "Control" o "Ctrl".
    El vídeo se descargará autimaticamente, el único inconviente es que no se puede guardar con un nombre elegido pero una vez terminada las descarga le podrás cambiar el nombre.
  • Usando Firefox: tienes que hacer un clic con el botón derecho del ratón, pero mientras mantienes presionada la tecla "Shift" del teclado.
    La tecla "Shift" es una tecla que suele estar situada en la segunda fila en los teclados, empezando por abajo. Suele estar tanto a la derecha como a la izquierda del todo de esta segunda línea. Es la tecla que se utiliza para poner una Mayúscula simple (por eso suele poner también "Mayus" y también suele estar representada por una flecha con sentido ascendente).
    El vídeo se descargará autimaticamente, el único inconviente es que no se puede guardar con un nombre elegido pero una vez terminada las descarga le podrás cambiar el nombre.
  • En otros navegadores: En algunos navegadores como por ejemplo Firefox permiten definir el comportamiento del mismo según que tipo de enlace intente abrir el usuario. Por ello podréis cambiar el funcionamiento del navegador para que cuando detecte un archivo MP4, en vez de intentar abrirlo con un plugin, lo descarge directamente, así solo se tendrá que realizar un clic normal rápido sobre el enlace.
    En Firefox es fácil y se hace en opciones de contenido. Allí tendréis que asociar la apertura de los archivos .mp4 (se puede hacer una búsqueda utilizando el cuadro para encontrarlo antes) su descarga directamente.
  • Para Internet Explorer y Opera: no conozco cómo se puede descargar un vídeo de manera más rápida que la normal (clic derecho y "guardar archivo como...") por lo que recomiendo para esos usuarios es, que utilicen, aunque sea solo para utilizar esta web, Google Chrome o Firefox. Y utilicen los métodos descritos más arriba.
  • De manera personal, recomiendo Google Chrome para el uso de esta web

Mi Tele descarga un archivo grande con estensión .php

Se ha detectado que en algunos navegadores, sobre todo móviles, en vez de descargarse un archivo .mp4 se descarga un .php. Si te fijas, el archivo es de gran tamaño, eso es por que sí es el vídeo, solo que se descarga con la extensión incorrecta. Con un navegador de archivos hay que cambiar el nombre del vídeo, reemplazando el .php por el .mp4.

¿Qué hago si PyDownTV no soporta un canal del cual quiero descargar un vídeo?

Si PyDownTV no soporta un canal que te gustaría que estuviera soportado, siempre puedes pedirlo, en la medida de lo posible, se intentará ayudarte. PyDownTV es software libre, y todo el mundo puede contribuír a desarrollar un canal, así que, ¿Por qué no pedirlo?
Puedes hacerlo en el siguiente enlace: Petición de canales.

  1. Crea una nueva Issue, haciendo clic a la derecha arriba en New Issue.
  2. En la derecha especifica que es un Pedir soporte para nuevo canal.
  3. Elige el título que creas conveniente, pero que sea descriptivo e incluya que es una petición de nuevo canal.
  4. Rellena el cuadro de la Issue con los detalles de la petición.
